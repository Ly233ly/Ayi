## 即梦提示词结构

### 核心公式

```
[风格/艺术家] + [主体描述] + [细节/材质] + [光影/色调] + [构图/视角] + [氛围/情绪] + [画质词]
```

### 即梦特色结构

```
[艺术风格参考]，[主体详细描述]，[材质/质感]，[色调]，[氛围描述]，[构图方式]，[画质增强词]
```

### 括号用法（说明性，非加权）

即梦中的括号主要用于**补充说明**，不是 SD 那种权重加权语法：

| 用法 | 示例 |
|------|------|
| 艺术家名注释 | `(Alfons Mucha)`、`(莫奈)` |
| 材质补充 | `(细线)`、`(毛孔/绒毛)` |
| 英文翻译 | `(hat)`、`(sash)` |
| 场景细节 | `(雨伞折射霓虹倒影)` |

**注意**：即梦不支持 `((关键词))` 双括号加权语法，那是 Stable Diffusion 的用法。在即梦中强调关键词，应该通过**重复使用**或**放在更前面**来实现。

## 具体细节描述技巧

### 色彩方案

不要只说"好看"，要给出具体色彩组合：

| 风格 | 色彩方案 |
|------|----------|
| 古风水墨 | 墨黑、灰白、淡青、浅绿 |
| 赛博朋克 | 紫蓝色、品红色、青蓝色、霓虹粉 |
| 高级感 | 暗金色、深琥珀色、黑色丝绒 |
| 梦幻 | 粉紫色、淡蓝色、柔和渐变 |
| 复古 | 暖黄色、褪色调、胶片颗粒感 |
| 暗黑 | 深灰色、暗红色、低饱和冷色调 |

## 常见问题

| 问题 | 解决方法 |
|------|----------|
| 风格不明确 | 加入具体艺术风格参考（如"极繁主义"、"Moebius风格"） |
| 细节不够 | 添加"极致细节"、"细腻"、"精致"等词，并给出具体色彩和材质描述 |
| 色调不对 | 明确写出具体色彩方案，如"紫蓝色与品红色交织"、"墨黑、灰白、淡青" |
| 构图平淡 | 指定"超广角构图"、"电影级构图"、"对角线构图" |
| 画质不佳 | 添加"8K"、"超高清"、"壁纸级精度" |
| 缺少氛围 | 加入"氛围感"、"梦幻"、"震撼"、"压迫感" |
| 重点不突出 | 使用括号加权：`((关键词))` 强调核心元素 |
| 过于笼统 | 具体化描述：不要"好看的颜色"，要"紫蓝粉交织的霓虹色彩" |
