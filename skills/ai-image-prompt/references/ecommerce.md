# 电商/商业风格提示词参考

> 适用于电商主图、详情页、促销物料、品牌包装等场景。

## 数据特征

- 平均长度：802 字符（最短的类型，精炼直接）
- 语言：中英混合（英文52%，中文41%）
- 核心词：brand, product, background, lighting, style, text, layout

## 常见结构模式

### 模式 1：中文简短广告（最常见的电商写法）

```
[品牌/产品名] + [核心卖点] + [视觉风格] + [氛围]
```

**示例**：
```
武则天同款皇后内衣广告大片，明黄底色龙纹刺绣，大红丝绸飘带衬托
```

```
赵飞燕同款掌中舞香膏广告，错金缠枝莲铜瓶，冷艳诡谲深宫氛围感，竖屏9:16
```

### 模式 2：英文品牌展示

```
[BRAND NAME] + [产品描述] + [展示方式] + [背景] + [光影] + [画质]
```

**示例**：
```
High-impact e-commerce infographic advertisement for "Fujifilm Instax Mini 11" blush pink instant camera. FOREGROUND: Extreme close-up of a hand holding a compact rounded blush-pink camera. CENTRAL MODEL: confident young woman. Background: soft gradient. Studio lighting, commercial photography style.
```

### 模式 3：品牌包装系统展示

```
[品牌名] + [类目] + [展示形式] + [包装细节] + [构图要求] + [画质]
```

**示例**：
```
请为品牌[品牌名称]、类目为[品牌类目]创作一张高审美、高完成度的「品牌包装系统展示图」。这不是普通电商白底图，而是一张具有品牌提案感、视觉识别系统感和商业摄影质感的包装家族陈列效果图。画面需要体现"同一品牌在多个包装载体上的统一落地"。4:5竖版海报构图。
```

| 构图 | composition, layout, center, focal point |

## 电商场景速查

| 场景 | 核心要素 | 常用表达 |
|------|----------|----------|
| 主图 | 产品居中+白底+光影 | "clean white background, studio lighting, product focus" |
| 详情页 | 多角度+细节+卖点 | "multiple angles, close-up details, feature highlights" |
| 促销海报 | 产品+文案+氛围 | "promotional poster, bold typography, festive atmosphere" |
| 品牌VI | 包装+品牌色+统一感 | "brand identity, packaging system, cohesive visual" |
| 直播封面 | 人物+产品+吸引力 | "livestream cover, engaging, eye-catching, promotional" |

## 完整示例

### 化妆品主图
```
Premium skincare product photography, [PRODUCT NAME] serum bottle in elegant glass packaging with gold cap, positioned on a marble pedestal. Soft pink gradient background with subtle floral elements. Studio lighting creating gentle highlights on the glass surface. Clean composition, luxury aesthetic, commercial product photography style. 4K resolution.
```

### 食品电商
```
Appetizing food product photography of [PRODUCT], beautifully plated on a ceramic dish. Fresh ingredients arranged around the main product. Warm studio lighting with soft shadows. Clean white background with subtle texture. Professional food styling, commercial photography, 4K resolution.
```

### 服装电商
```
Fashion product flat lay of [PRODUCT NAME], neatly arranged on a clean white surface. Garment folded or styled naturally. Accessories placed beside for context. Soft natural lighting from above. Minimal composition, editorial style, clean aesthetic. Commercial fashion photography, 4K resolution.
```
