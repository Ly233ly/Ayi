# 分析方法论与模块参考

## 五层递进分析框架

整个分析沿"钱花到哪里 → 花得是否合理 → 未来会怎样 → 怎么改善"主线推进：

```
第一层·结构视角 ──→ 钱去哪了？（收支全景 + 吃喝拆解）
第二层·效率视角 ──→ 花得值不值？（收入质量 + 支出效率 + 理财识别）
第三层·行为视角 ──→ 习惯好不好？（现金流时序 + 消费画像）
第四层·预测视角 ──→ 未来怎么样？（双情境推演 + 省钱量化）
第五层·诊断视角 ──→ 综合仪表盘（7项PASS/FAIL + 健康评分）
```

## 十段深度分析模块

| # | 层级 | 模块 | 分析角度 | 核心逻辑 |
|---|------|------|---------|---------|
| 一 | 结构 | 收支结构全景 | 三分法分级 | 每笔支出归入必要/改善/欲望，对照50/30/20法则 |
| 四 | 结构 | 吃喝行为拆解 | 恩格尔系数 | 7子类明细/次均/日均，判断恩格尔系数是否合理 |
| 二 | 效率 | 收入深度分析 | 稳定性+多元化 | CV变异系数+来源数，评估抗风险能力 |
| 三 | 效率 | 支出效率分析 | 固定vs可变 | 固定/收入比 + 大额剔除后日均 + 周期性消费识别 |
| 七 | 效率 | 理财与资产 | 消费vs投资 | 理财类自动识别，剔除后算真实消费率 |
| 五 | 行为 | 现金流时序 | 发薪效应 | 发薪后7天冲动检测 + 上下半月分布均匀度 |
| 六 | 行为 | 消费行为画像 | 场景还原 | 5时段分布 + 工作日/周末 + 7档金额区间 |
| 八 | 预测 | 未来预测 | 双情境推演 | 情景A(正常月) / 情景B(含大额)，预测3个月净资产 |
| 九 | 预测 | 省钱潜力 | 量化改善 | 逐项优化目标+月省金额，算优化后储蓄率 |
| 十 | 诊断 | 指标仪表盘 | 7项硬指标 | 结余率/固定比/恩格尔/欲望比/记账率/必要比/收入稳定性 → PASS/FAIL + 健康分 |

> 注：编号保留原有顺序以兼容脚本输出，层级列体现五层递进关系。

## HTML 报告

使用 `--html` 生成网页报告：

```bash
python3 personal-finance/scripts/analyze.py 账单.csv --html 报告.html
```

HTML 采用**模板分离**设计 — `scripts/template.html` 包含所有结构和样式，脚本只负责填入数据。修改模板无需触碰 Python 代码。
