# 写实/产品风格提示词参考

> 适用于超写实摄影、产品展示、材质细节等场景。

## 数据特征

- 平均长度：1553 字符（最长的类型）
- 语言：英文为主（71%），中文（7%），JSON（22%）
- 核心词：photorealistic, texture, natural, lighting, background

## 常见结构模式

### 模式 1：模板占位符式（品牌类最常见）

```
[BRAND NAME] + [产品描述] + [固定结构] + [占位符变量]
```

**示例**：
```
[BRAND NAME] | [COLOR]

Create a hyper-realistic [PRODUCT TYPE] with [MATERIAL DETAILS]. The product should feature [BRAND LOGO] prominently, with [COLOR SCHEME] and [TEXTURE DETAILS]. Studio lighting, clean background, commercial photography style.
```

### 模式 2：详细材质描述式

```
[产品名称] + [材质细节] + [表面质感] + [光影效果] + [背景] + [画质]
```

**示例**：
```
Create a hyper-realistic 3D render of a large capsule-shaped container. The top half is solid and glossy in the brand's signature color, featuring the official [BRAND] logo prominently. The bottom half is transparent, revealing multiple miniature products inside. Soft studio lighting, clean white background, ultra-detailed texture, 8K resolution.
```

## 即梦中文写实模式

> 基于 1290 条即梦写实数据分析

### 常见结构
```
[写实风格]，[主体描述]，[材质/质感]，[光影效果]，[背景]，[构图]，[画质词]
```

### 写实高频词
- 写实度：写实、真实、照片级、高清、超清
- 材质：细腻、精致、质感十足、纹理清晰
- 光影：质感光照、自然光线、明暗对比强烈
- 画质：8K、超高清、壁纸级精度、极致细节

### 写实画质组合
```
写实摄影风格，质感光照，自然光线，质感十足，电影级别镜头，高清超高清画质
```
