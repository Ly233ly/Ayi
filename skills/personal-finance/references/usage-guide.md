# 使用指南

## Python Tool

1 个脚本，纯 stdlib，零依赖：

```bash
python3 personal-finance/scripts/analyze.py --help
```

## 列名自动匹配

脚本会自动识别以下列名变体：

| 语义 | 支持的列名 |
|------|-----------|
| 时间 | `时间`, `time`, `日期`, `date`, `交易时间` |
| 类型 | `类型`, `type`, `收支类型`, `方向` |
| 分类 | `分类`, `category`, `一级分类`, `大类` |
| 二级分类 | `二级分类`, `subcategory`, `子分类`, `小类` |
| 金额 | `金额`, `amount`, `数额`, `价格` |
| 备注 | `备注`, `note`, `说明`, `描述`, `remark` |

不匹配时脚本会列出可用列名并尝试继续。

## 常见场景

| 用户说 | 执行 |
|--------|------|
| "分析账单" / "看看花了多少" | 运行 analyze.py → 呈现十段数据概览 |
| "深度分析" / "全面分析" / "深入看看" | 先跑 analyze.py → 加载 `references/deep-analysis-prompt.md` → 按 8 模块输出深度报告 |
| "生成报告" / "导出 HTML" | 运行 analyze.py --html 报告.html |
| 拖入 CSV 文件 | 运行 analyze.py 该文件 |
| 目录下有多个 CSV | 先问用户分析哪个 |

## 深度分析标准参数

- 年化收益率：8%
- 复利年限：10年 / 20年 / 30年
- 月工作日：22 天
- 日工作时长：8 小时
